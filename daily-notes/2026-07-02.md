# 2026-07-02 — Day 2

## Threat Intel Reading

### FortiBleed → INC & Lynx Ransomware (SOCRadar)
- 430,000 FortiGate firewalls targeted globally
- 110 million credentials stolen via Golang packet sniffer
- 12,000 devices had sniffer installed passively on network traffic
- Operator found logged into both INC Ransom and Lynx panels simultaneously
- Confirms Initial Access Broker model: steal access → sell to ransomware groups
- ~20 person organized operation, Russian-speaking
- Have an unpatched Nextcloud zero-day
- Lesson: never expose firewall admin panels to the internet

### ChocoPoC — Malware Targeting Security Researchers (YesWeHack/Sekoia)
- Fake PoC repos on GitHub targeting researchers who test CVEs
- Malware hidden in dependency chain, not the PoC itself
- Chain: frint → skytext → gradient.so → trojan
- Only activates when EXPLOIT_POC.py is present — bypasses sandboxes
- Steals browser passwords, cookies, shell history, files
- C2 hidden inside Mapbox API calls using domain fronting
- Targets: FortiWeb, PAN-OS, Ivanti, Check Point CVEs
- Lesson: always read full dependency chain, always use throwaway VM

### JADEPUFFER — First Fully Autonomous AI Ransomware (Sysdig)
- AI agent ran entire attack start to finish with no human at keyboard
- Entry: CVE-2025-3248 Langflow unauthenticated RCE
- Swept for API keys, cloud creds, crypto wallets, database logins
- Raided MinIO with default password minioadmin:minioadmin
- Encrypted 1,342 Nacos settings, dropped ransom note, deleted databases
- Generated encryption key, printed once, never saved — victim cannot recover data
- Code had plain English comments explaining each step — confirmed AI authorship
- Fixed its own mistakes in 31 seconds
- Lesson: default credentials and exposed admin interfaces are now AI targets
