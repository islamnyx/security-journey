## PortSwigger Labs

### Lab 1 — API Documentation Exploitation
- Found API docs at `/api` endpoint
- Discovered DELETE /user/[username] endpoint
- Used Burp Repeater to call: DELETE /user/carlos
- Got 200 OK response, user deleted
- Concept: exposed API documentation reveals attack surface directly

### Lab 2 — HTTP Method Override (Leather Jacket)
- Found `/product?productId=1` endpoint normally accepts GET
- Discovered hidden `/api/products/1/price` endpoint via Proxy history
- Used OPTIONS method to discover allowed HTTP methods (GET, PATCH)
- Changed HTTP method from GET to PATCH
- Added Content-Type: application/json header
- Sent body: {"price":0}
- Successfully modified product price from $1337 to $0
- Purchased jacket for $0
- Concept: APIs often expose dangerous methods (PATCH, PUT, DELETE) not documented in UI

## Threat Intel Reading

### PolinRider — North Korean Supply Chain Attack (Socket/OpenSourceMalware)
- Compromised 1,951 GitHub repositories across 1,047 unique owners
- 108 unique malicious packages: 19 npm, 10 Composer, 61 Go modules, 1 Chrome extension
- Uses Contagious Interview campaign — fake job recruiters to compromise developers
- Attack chain: fake job offer → malware → GitHub access → plant obfuscated JS → rewrite Git history
- Hides malicious code in config files (postcss.config.mjs, tailwind.config.js, eslint.config.mjs)
- Executes via VS Code tasks with "runOn: 'folderOpen'" trigger
- Final payload: DEV#POPPER RAT + OmniStealer stealing credentials and files
- Git history rewriting makes commits appear legitimate and older

### Kairos — Data Extortion Without Encryption (Ransom-ISAC)
- Stole files from Union County, Ohio (~70,000 residents)
- No encryption — pure data extortion model
- Demanded $3M → negotiated down to $1M payment (9.44 BTC)
- Sent "proof of deletion" but no actual guarantee data was deleted
- Shows shift: only ~50% of ransomware attacks now use encryption
- Got in via weak password (guessed credentials)
- Defense: MFA, monitor outbound data transfers, segment networks
